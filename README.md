# Clean monorepo Template with Vue, Ionic, and Capacitor on Nuxt 4

<center> 
<img src="assets/cover.png" alt="android and ios emulator">
</center>

- Nuxt 4
- Ionic 8
- Vue 3.5
- Capacitor 7
- Typescript 5.7
- Tailwind CSS 4

## Installation

```bash
pnpm install
```

## Development HMR

Go to app folder `cd ./packages/app` and run the following command:

```bash
# pwa
pnpm dev

# ios
pnpm ionic cap run ios

# android
pnpm ionic cap run android
```

## Build

```bash
pnpm build
```
